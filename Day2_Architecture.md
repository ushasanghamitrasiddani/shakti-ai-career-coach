# Shakti AI Architecture

## Goal

Build an AI-powered Career Restart Coach that helps women returning to work after a career break.

The system should:

- Assess user skills and experience
- Identify transferable skills
- Generate personalized learning roadmaps
- Create professional resumes
- Conduct mock interviews
- Recommend suitable job roles
